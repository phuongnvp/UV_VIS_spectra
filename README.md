UV-VIS spectra prediction using AttentiveFP.
We also develop a web-based application using the Streamlit framework. The interface allows users to input molecular structures as SMILES strings and obtain predicted UV-Vis spectra in range of 220-700 nm. 
In addition to spectral visualization, the predicted data can be exported in CSV format. Attention weight information is also provided for interpretation. 
The application is publicly available at https://spectra-prediction.streamlit.app/.
